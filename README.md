# TANET 間接提示注入研究工作區

本工作區只保留研究計畫、文獻、資料來源、實驗環境、可重現程式與有效實驗結果。

| 位置 | 用途 |
|---|---|
| `rag-ipi-defense/` | 主要程式、測試、研究文件與實驗結果 |
| `research plan/` | 研究計畫原稿 |
| `literature review/` | Prompt injection、RAG 防禦與 agent security 文獻 |
| `BIPIA/` | 上游 BIPIA benchmark source |
| `datasets/` | NewsQA、XSum、JailbreakBench 等外部資料 |
| `env/` | 可重現的本機 Python 執行環境 |

主要進度與最新數字見 `rag-ipi-defense/docs/experiment-status-2026-07-04.md`。大型外部資料與模型快取不推送 Git；最新主實驗、manifest、參數與研究紀錄則保留在本機工作區。
